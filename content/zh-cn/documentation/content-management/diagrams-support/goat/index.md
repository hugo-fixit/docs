[[GoAT][goat]]^(Go ASCII Tool) 是 [markdeep.mini.js][markdeep] 图像生成器的 Go 语言实现。

<!--more-->

## 用法 {#usage}

要使用 GoAT，只需将 ASCII 输入放在代码块中，并将语言设置为 `goat`。

````markdown
```goat
// ASCII input here
```
````

以下是使用 GoAT 生成 SVG 的一些示例。

查看更多[示例][goat-examples] 。
